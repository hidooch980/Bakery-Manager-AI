# Bakery Manager AI v1.1.0 Stable Update

## امکانات اضافه‌شده
- اپلیکیشن موبایل آفلاین‌محور و پایدار برای نصب و استفاده روزانه
- داشبورد مدیریتی کامل
- ثبت فروش نقدی، کارت‌خوان و نسیه
- ثبت تولید با آرد، خمیر، چانه، نان، ضایعات و مسئول تولید
- کنترل آرد و انبار با موجودی لحظه‌ای
- ثبت هزینه‌ها
- گزارش روزانه سود، فروش، هزینه، تولید و موجودی آرد
- خروجی CSV برای حسابداری
- دستیار AI برای هشدار کمبود آرد، سود منفی، تولید/فروش ثبت‌نشده و بازده پایین
- پیش‌بینی تولید فردا بر اساس فروش‌های اخیر
- تنظیمات نانوایی، قیمت نان، وزن کیسه، حداقل آرد و ظرفیت تولید
- ورود آفلاین پیش‌فرض در صورت قطع سرور
- صف همگام‌سازی آفلاین برای داده‌های ثبت‌شده
- تغییر applicationId به `com.hidooch980.bakerymanagerai`

## ساخت APK
در این محیط Flutter نصب نیست، بنابراین APK مستقیم ساخته نشد. برای ساخت:

```bash
cd mobile
flutter pub get
flutter analyze
flutter build apk --release --dart-define=API_BASE_URL=http://185.97.118.255:3000
```

## خروجی
```text
mobile/build/app/outputs/flutter-apk/app-release.apk
```
