# Bakery Manager AI — Frontend

پنل وب مدیریت نانوایی، ساخته‌شده با **React 19 + TypeScript + Vite**.

## اجرا

```bash
npm install
npm run dev
```

آدرس پیش‌فرض سرور API: `http://localhost:3000`

## دستورات

| دستور | توضیح |
|---|---|
| `npm run dev` | اجرای محیط توسعه |
| `npm run build` | ساخت نسخه production |
| `npm run lint` | بررسی کد با oxlint |
| `npm run preview` | پیش‌نمایش نسخه build شده |

## صفحات اصلی

داشبورد، فروش، شیفت فروش، تولید، نوع نان، کنترل وزن چانه، خرید آرد، انبار، صندوق، حقوق و دستمزد، درآمد و هزینه، دوره نانینو، کارکنان، کاربران و تحلیل‌ها.
